# 454 Project

A secure, cloud-deployed web application for sharing and discussing SoundCloud tracks with role-based access control, audit logging, and compliance-focused security.

## Tech Stack

- **Frontend:** React 18, Vite, React Router, Axios
- **Backend:** Python 3.11, FastAPI, SQLAlchemy, Alembic
- **Database:** PostgreSQL 15
- **Infrastructure:** AWS (ECS Fargate, RDS, S3, CloudFront, Lambda, WAF)
- **IaC:** Terraform
- **Containerization:** Docker, Docker Compose
- **Auth:** JWT (PyJWT), bcrypt
